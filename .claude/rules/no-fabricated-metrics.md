# Never Fabricate Metrics

## Hard Rule

When creating resumes, professional content, presentations, or any material that makes claims about the user's work:

**NEVER invent, inflate, round up, or extrapolate metrics.** Only include claims directly evidenced from the user's actual work.

## If No Evidence Exists

Use qualitative descriptions:
```
WRONG: "Reduced processing time by 60%" (invented number)
RIGHT: "Significantly reduced processing time" or "ask user for actual metric"
```

Better yet: ask the user for the real number.

## Why This Is Critical

Fabricated metrics in professional content:
- Destroy trust if caught
- Are especially ironic when the content is about integrity and capability
- Cannot be defended in interviews ("Where does that number come from?")
