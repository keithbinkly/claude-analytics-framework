## Quick Start Examples

### Example 1: Dynamic Column Selection

```sql
-- Instead of hardcoding columns
SELECT
    {{ dbt_utils.star(
        from=ref('users'),
        except=['_fivetran_synced', 'updated_at']
    ) }}
FROM {{ ref('users') }}
```

### Example 2: Pivot Payment Methods

```sql
{% set payment_methods = dbt_utils.get_column_values(
    table=ref('payments'),
    column='payment_method'
) %}

SELECT
    order_id,
    {{ dbt_utils.pivot(
        column='payment_method',
        values=payment_methods,
        agg='sum',
        then_value='amount'
    ) }}
FROM {{ ref('payments') }}
GROUP BY 1
```

### Example 3: Create Surrogate Key

```sql
SELECT
    {{ dbt_utils.generate_surrogate_key([
        'user_id',
        'order_id',
        'timestamp'
    ]) }} as unique_key,
    *
FROM {{ ref('orders') }}
```

### Example 4: Union Similar Tables

```sql
{% set event_tables = dbt_utils.get_relations_by_pattern(
    schema_pattern='raw_events',
    table_pattern='events_%'
) %}

{{ dbt_utils.union_relations(
    relations=event_tables,
    exclude=['_loaded_at']
) }}
```
